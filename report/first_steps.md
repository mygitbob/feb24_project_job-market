## Data found on various locations
### Adzuna
#### API Documentation
https://developer.adzuna.com/overview
we can get historical salary data with the api

**web interface**:
- location
- company (no profiles)
- estimated salary (for jobs in US, missing for example for jobs in Germany)
- job category
- date posted
- salary
- remote/flexibel
- employment type
- hours
